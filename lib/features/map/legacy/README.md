# Legacy Flutter Map Implementations

## Purpose

This folder is reserved for **working reference implementations** using `flutter_map` with raster tiles, as a stable baseline that works across all platforms.

## Current map stack

The app uses **Google Maps** for all map rendering. The `AdaptiveMapWidget` may still support a flutter_map raster fallback in some code paths. This folder is for legacy reference only; no Mapbox migration is in scope.
